## 참조링크


### 변수와 상수
+ var
  변수를 선언할 때 사용되는 키워드 이며, var 로 선언한 변수는 값을 변경할 수 있다.
+ val
  상수를 선언할 때 사용되는 키워드 이며, val 로 선언한 상수는 값을 변경할 수 없다.(Java에서의 final)
  ```KOTLIN
  var a:Int = 100  //변수의 선언
  val b:Int = 200  //상수의 선언

  a += 1  //var이기 때문에 변경 가능
  b += 1  //val이기 때문에 변경 불가능
  ```

### 함수(메서드)
+ 형식
  fun 함수명 (인자1: 자료형1, 인자2: 자료형2, ...): 반환 자료형
  ```KOTLIN
  fun welcome (msg : String) : Unit {
      println("$msg, welcome !!")
  }
  welcome("안녕하세요~ ")          //출력 결과 : 안녕하세요~ , welcome !!
  ```

### 기본자료형
+ 숫자형(Double / Float / Long / Int / Short / Byte)
  기본적으로 자료형을 알아서 맞춰주며, 10.0 을 입력하면 Double로 받아들임 굳이 10.0을 float형으로 받기위해선 10.0f 처럼 해줘야함
  ```KOTLIN
  var a:Int       = 10
  var b:Long      = 10L
  var c:Double    = 10.0
  var d:Float     = 10.0f

  var aa = 10         // Int
  var bb = 10L        // Long
  var cc = 10.0       // Double
  var dd = 10.0f      // Float
  ```
